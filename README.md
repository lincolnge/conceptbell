Concept Bell
========

运行:
--------

	gulp serve

build:  

	gulp


app/ 这个文件夹是放代码  
dist/ 这个文件夹是放编译完后的代码  

自动化工具未真正完成前提下, 手动将 `main.min.js`, `index.css`, `index.html` 合在一起, 放在根目录 `index.html`文件内.
